# 🎉 Aplicación Web Interactiva — Quinta Inés María BED, Catering y Eventos

## 📋 Descripción
Sistema web interactivo para la planificación y personalización de eventos de la 
Quinta Inés María BED, Catering y Eventos, ubicada en el cantón Chambo, 
provincia de Chimborazo, Ecuador.

Permite a los clientes explorar servicios, configurar su evento de forma 
personalizada y obtener cotizaciones automáticas en tiempo real, sin necesidad 
de contacto presencial previo.

## 📦 Módulos del Sistema

1. 🖼️ **Galería Multimedia** — Exploración de eventos anteriores
2. 📋 **Catálogo de Paquetes** — Bronce, Silver, Gold, Corporativa, Alfombra Roja
3. ⚙️ **Configurador Interactivo** — Personalización paso a paso en tiempo real
4. 💰 **Cotización Automática** — Generación de presupuestos en PDF
5. 📨 **Gestión de Solicitudes** — Panel de administración de reservas
6. 👤 **Autenticación y Perfiles** — Registro y acceso de usuarios
7. 🔧 **Administración** — Panel de gestión de contenidos
8. 🔔 **Notificaciones** — Envío automático de correos
